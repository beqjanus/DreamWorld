Text to speech

 

How to make Wav and MP3 files you can play in-world


DreamGrid contains a text to speech converter that can make wave and mp3 files from text. These files can be used in-world by uploading the wav file, or by playing either wav or mp3 using media on a prim.

This feature is still in development but the basics shown here will remain unchanged. An API is will allow LSL scripts to request Mp3 files by name for media-on-a-prim use so that products can talk in-world, without being uploaded as wav files.

Requirements

You must have a sound card to use this feature. VPS Servers typically do not have sound cards. Physical servers as well. There are several solutions:

         If you have physical access to the server, plug in a USB sound card.

         If you do not have access, you can install a virtual sound card.

 

GUI

This GUI makes the files.

Windows 10 has 4 speech engines including two that are enabled by DreamGrid. Windows 7 comes with just one. You can add additional SAPI5 voices by purchasing them.

Any line beginning with M: will speak in the male voice. F: will speak in a female voice. Any line without a letter will be spoken in the selected voice.
