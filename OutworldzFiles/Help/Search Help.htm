Search Help

 

Search has built-in pattern matching for making string comparisons. The following table shows the wildcard characters you can use and the number of digits or strings they match.

WILDCARD CHARACTERS USED IN STRING COMPARISONS

Character(s) in pattern

Matches in expression

?

Any single character

*

Zero or more characters

#

Any single digit (09)

[ charlist ]

Any single character in charlist

[! charlist ]

Any single character not in charlist
