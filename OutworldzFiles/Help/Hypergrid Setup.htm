Hypergrid Setup

Connecting to the Hypergrid  is enabled by default. There are several steps you must take to enable the your network to support it.

 

UPnP

Enable UPnP – This switch is located in Network Ports. If enabled, the server will attempt to connect to your router and set up all the ports. This does not always work. If this switch is on after running Dreamgrid, UPnP does work and you will need to try step 2.

 



Manually Forward Ports in your Router


Go to your router login page and get logged in. Look for a page for Port Forwarding.

A list of routers and instructions is at http://www.portforward.com which can help once you identify your router type.

 

Get your PC LAN IP address by going to a DOS prompt and typing 'ipconfig'.

In the above photo, my IP was 192.168.1.3.   Yours is likely to be in the range 192.160.*.*, but it could also be a 10.0.*.* number.

 

For my Linksys, it looks like this.

 


Which ports? You need to Port-Forward 80, 8002, and 8004-8010 (or higher, I usually open up to 8050) to have room for expansion. You can also add 8080 and 8081 for Shoutcast/Icecast. 

 

Do not do 8003 for security reasons!

 

Usually these support ranges, such as 8004-8010. The latter ones are for regions - you need one port per region, so this would let you have 7 regions. Add more if you want - nothing wrong with opening to 8020 or 8030 or so.  In the above photo, I have done ports 8004 through 8005.

 

Ports need to be open for both TCP and UDP.

Your router will have a button or a pulldown to select UDP, TCP, or both.  If you do not have a Both or All option, you will have to add them twice, once for UDP and once for TCP.

 

How do I know this worked? You can tell if it worked by starting Opensimulator, and then use a web browser to go to http://www.canyouseeme.org, and enter 8002 when Opensimulator is running. If that works, the ports are fine.

 

 

Loopback

 

The second part that must work is loopback - either your router supports it, or not. You can at once tell if your loopback works after running the above CanYouSeeMe.org tests.

You use your web browser to navigate to http://(YOUR PUBLIC IP):8002.  This IP is availab le from http://Canyouseeme.org. Mine was http://24.173.0.66:8002. Of course, your IP address will be different than shown here, so use whatever CanYouSeeme shows you. If the system is working, you get a web page.

 

If it works, go to the Settings->Expert Settings menu, and click the Hypergrid DNS Name Button.

 

Click it and you get a pop up, where you can enter your desired DNS name

 

Either click Next Name until you see one you like, and press Save, or type a name like 'Whatever.Outworldz.net". The name will not work with spaces, and no funny characters. It uses letters, numbers and a dash (-) only. Then click Save, and restart Opensim.

 

You use your web browser to navigate to http://(YOUR NEW NAME):8002, such as http://Whatever.Outworldz.net:8002.  If you get the web page, use that same name to add the Grid Settings to your viewer, and you should be set to go.
